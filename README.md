# Arduino_Control-Robot-from-PC

This is an Arduino project to command a robot with a python GUI with an USB cable
<br><br>
Dependencies<br>
-python2.7
-python-glade2
-python-serial

Debian and Ubuntu<br>
sudo apt-get install python2.7 python-serial python-glade2
